# Portfolio Website

## Features

- Responsive Design
- Three Pages
  - Home
  - About
  - Contact
- Flexbox & CSS Grid
- Hamburger Menu
- Hover Effects
- Mobile Friendly

## Technologies Used

- HTML5
- CSS3

portfolio-site/

## How to Run

1. Download the project.
2. Open `index.html` in your browser.
3. Navigate between pages using the navigation bar.
